khi thiết kế một class thì cần cần nhắc chỉ thêm thuộc tính khi mà ta cần thuộc tính đó làm cái gì đó
ví dụ string ID và string name, nếu ta cần ID để tra thông tin hoặc name để hiện thị thông tin, 
nếu có một thuộc tính là bool gender mà không sử dụng nó để làm gì hết thì không cần thêm vào
ví dụ như cần đếm xem đọc giả có bao nhiêu là nam, bao nhiêu là nữ thì có thể thêm bool gender vào
